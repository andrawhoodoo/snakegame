Class: WorldModel

WorldModel

Model Class representing a virtual world for a Snake.

new WorldModel(aca, w, h)

Creates a world model and ensures that an instance of Snake class is passed. Also initializes a views, actors arrays

Parameters:
Name Type Description
aca ActorCollisionHandler

allows for interactions between the world's classes.
w int

width of desired world view. If no int passed, default to 100.
h int

height of desired world view. If no int passed, default to 100.

Moves the snake belonging to the WorldModel, checks for collisions between actors. Updates the display of CanvasView if there is a paired CanvasView class with this WorldModel.
